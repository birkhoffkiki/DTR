## Usage

### Install
- Clone this repo:

```bash
git clone https://github.com/birkhoffkiki/DTR.git  
cd WORK_DIRCTORY
```
- create environment  
```bash
conda create --name DTR python=3.9
conda activate DTR
pip install -r requirements.txt
```
